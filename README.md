# MSTeamsClone
make realtime video chat website

## Dependencies
Run below commmands to install dependencies\
npm i express ejs socket.io\
npm i uuid\
npm i --save-dev nodemon\
npm i -g peer\
npm i mongoose

## Run on local server
npm run devStart\
In new terminal\
peerjs --port 3001\
go to port 3000 of localhost
